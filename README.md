A graphical user interface for [BOLDconnectR](https://doi.org/10.32614/CRAN.package.BOLDconnectR) allowing basic exploration and analysis of data from BOLD.

**BOLDreflectR** is a Shiny app built in R, and packaged for use as a desktop application using a [fork](https://github.com/skmonckton/nhyris) of [nhyris](https://github.com/jahnen/nhyris). This repository includes the necessary files to build the packaged app (currently only for Windows). If desired, the R code can be downloaded from the Shiny sub-directory ([skmonckton/BOLDreflectR/tree/main/shiny](https://github.com/skmonckton/BOLDreflectR/tree/main/shiny)) to launch the app from within RStudio.
