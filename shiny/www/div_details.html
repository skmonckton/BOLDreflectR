Diversity statistics are computed using the following methods:

AnalysisR packageFunctionNotes
Community matrixBOLDconnectR
bold.analyze.diversity()
Preston abundance distributionvegan
prestondistr()
Called via BOLDconnectR.
Shannon indexvegan
diversity()
Called via BOLDconnectR.
Rarefaction & extrapolationiNEXT
iNEXT()
Input parameters: 
endpoint = N * 2, knots = 40, se = TRUE, conf = 0.95, nboot = 50
.
For individual-based rarefaction: 
datatype = "abundance"
, and input data are abundance counts of taxa.
For incidence-based rarefaction (i.e. presence-absence): 
datatype = "incidence_freq"
, and input data are incidence frequencies generated from the community matrix using 
iNEXT::as.incfreq()
Beta diversity indexBAT
beta()
Called via BOLDconnectR.
Beta diversity ordination (NMDS)vegan
monoMDS()
Input parameters: 
k = 2, maxit = 600
.

All analyses are computed using available data from the Data tab with current filters applied. Input data are further filtered to exclude records lacking a species/BIN or site/coordinates, except in the case of individual-based rarefaction of overall data, which includes records without site/coordinates.
